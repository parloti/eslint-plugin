## 🧱 Changes

- File: src/application/**
  - Moved package-level rule aggregation and the `customRules` registry into a dedicated application layer.
- File: src/domain/index.ts
  - Added a domain barrel so application code can depend on a single domain entry instead of internal module paths.
- File: src/infrastructure/index.ts
  - Added an infrastructure barrel for plugin and preset exports and re-exported the application-owned `customRules` registry.
- File: src/index.ts
  - Kept the package public surface stable while routing exports through the new application and infrastructure barrels.
- File: eslint.config.ts
  - Reworked the repository boundaries model to `entrypoint`, `application`, `infrastructure`, and `domain` with inward-only dependency rules.
- File: README.md, docs/design/architecture.md, .github/copilot-instructions.md
  - Updated contributor-facing and agent-facing docs to describe the new layer layout accurately.
